# Contribuir a Prospería

Gracias por ayudar a que la organización financiera sea más accesible.

1. Abre un issue breve que describa a la persona y el problema.
2. Crea una rama desde `main`.
3. Mantén el lenguaje sencillo y sigue `AGENTS.md`.
4. Ejecuta pruebas y compilación.
5. Abre un pull request con pasos para probar el cambio.

No incluyas información financiera real, correos, tokens o archivos `.env` en issues, commits o capturas.
