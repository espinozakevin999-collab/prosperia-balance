# Demo de Prospería Balance — 2:35 minutos

## Antes de grabar

- Usar la aplicación en una ventana limpia y con zoom de 100 %.
- Tener preparados dos movimientos de ejemplo: una venta de $500 y un gasto de insumos de $120.
- Iniciar sesión para mostrar la recomendación con GPT-5.6 cuando ya esté activa.
- Grabar a 1080p y comprobar que el cursor y el texto se vean con claridad.

## Guion

### 0:00–0:15 — El problema

**Pantalla:** portada de Prospería y después la aplicación en Inicio.

**Voz:**

> Muchas personas y pequeños negocios quieren organizar su dinero, pero las aplicaciones financieras suelen ser complicadas. Prospería Balance lo reduce a tres preguntas: cuánto entró, cuánto salió y cuánto quedó.

### 0:15–0:35 — La propuesta

**Pantalla:** recorrer las tres tarjetas del resumen y los datos de ejemplo.

**Voz:**

> Diseñamos Prospería para personas con poca experiencia financiera o digital. Usa palabras cotidianas, botones grandes y datos de ejemplo para aprender sin miedo. Funciona desde el navegador en celular y computadora.

### 0:35–1:05 — Registrar movimientos

**Pantalla:** pulsar **Empezar con mis datos**. Registrar una venta de $500 y después un gasto de insumos de $120.

**Voz:**

> Registrar un movimiento toma pocos pasos. Anoto una venta de quinientos pesos y un gasto de insumos de ciento veinte. El resumen se actualiza inmediatamente y muestra que quedaron trescientos ochenta pesos.

### 1:05–1:25 — Entender sin tecnicismos

**Pantalla:** mostrar categorías, pistas y la sección **Movimientos**.

**Voz:**

> Prospería explica en qué se fue el dinero y propone una acción pequeña. También permite buscar, corregir o borrar movimientos. Las funciones esenciales trabajan localmente, sin cuenta y sin inteligencia artificial.

### 1:25–1:50 — Privacidad y sincronización

**Pantalla:** abrir **Tu cuenta** y mostrar las opciones de respaldo, presupuesto y CSV.

**Voz:**

> Si la persona crea una cuenta, Supabase sincroniza sus datos de forma privada entre dispositivos. Las políticas de seguridad separan los registros de cada usuario. También puede fijar un límite mensual e importar o descargar sus movimientos.

### 1:50–2:15 — GPT-5.6

**Pantalla:** abrir **Análisis** y pulsar **Mejorar con IA**; mostrar una respuesta real de GPT-5.6 Terra.

**Voz:**

> GPT-5.6 Terra convierte un resumen mínimo en una observación y una acción concreta. Nunca recibe el correo, las notas ni la descripción de movimientos individuales. Si la IA no está disponible, Prospería conserva recomendaciones locales útiles.

### 2:15–2:35 — Codex y cierre

**Pantalla:** alternar brevemente entre el repositorio público, las pruebas y la aplicación móvil.

**Voz:**

> Codex nos ayudó a transformar el prototipo en una aplicación funcional: interfaz, cálculos, sincronización, pruebas, accesibilidad, documentación y seguridad. Prospería Balance busca democratizar una herramienta esencial: entender el propio dinero sin enredos.

Prospería Balance es una herramienta financiera accesible para personas y microemprendimientos. Fue desarrollada con Codex y usa GPT-5.6 Terra para ofrecer orientación breve a partir de cifras agregadas, manteniendo las funciones esenciales disponibles sin IA.
